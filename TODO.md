Shrink/grow image:

```sh
# unmount all before
PRI_DISK_ROOT=$(blkid | grep 'LABEL="/"' | sed 's/:.*$//')
echo "PRI_DISK_ROOT=$PRI_DISK_ROOT"
# check
e2fsck -f "$PRI_DISK_ROOT"
# shrink
resize2fs -M "$PRI_DISK_ROOT"
# grow
resize2fs "$PRI_DISK_ROOT"
```
